Project Setup & Usage

Install

Run npm install to install dependencies.

Start the server

After installing, run npm start to launch the Express app.

Open http://localhost:3000 in your browser.

Admin dashboard

Navigate to /admin.html to enter your API keys. This sets up environment variables so the server can communicate with OnlyFans and OpenAI.

Features

  • Full Sync (A-1) fetches all fans, messages and purchases.
  • Quick DM (B-1) compose a direct message.
  • Mass Blast (B-2) send or schedule a bulk DM.
  • Queue View (B-3) list scheduled posts and messages.
  • Vault Browser (B-4) manage media lists.
  • Payout Pulse (B-5) display your available balance.
  • High Spend Tracker (B-6) flags big tippers nightly.
  • A/B Variant Lab (B-9) run messaging experiments.
  • Auto Thank (B-10) automatically thank purchases.
